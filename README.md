# astrbot_plugin_lyn

AstrBot 综合工具插件，提供 JM PDF 下载、JM 搜索、Pixiv 图片私信、视频链接信息解析和视频下载功能。

## 功能

- `/jm <漫画ID>`：下载 JM 漫画，转换为 PDF 后发送文件。
- `/jm搜索 <关键词1> <关键词2>...`：搜索 JM 漫画，展示前 10 条结果及作者、标签等信息。
- `/pixiv <PixivID/链接>`：通过 `pixiv.re` 获取 Pixiv 作品图片，并私信给触发用户。
- `/下载 <视频链接/小程序>`：使用 `yt-dlp` 下载视频，并作为文件发送。
- `/视频 <视频链接/小程序>`：使用 `yt-dlp` 下载视频，并作为视频消息发送。

## Pixiv 说明

- 当前仅支持 QQ OneBot v11 平台私信发送。
- 支持 Pixiv 数字 ID、`pixiv.re` 图片链接、Pixiv 作品页链接和带 `illust_id`/`id` 的链接。
- 最多探测 500 页图片。
- 私信使用合并转发发送，每条聊天记录最多 42 张图片；超过 42 张会自动拆成下一条聊天记录继续发送。

## 视频链接解析

普通消息中出现支持的视频平台链接或 QQ 小程序卡片时，插件会自动解析媒体信息并回复。自动解析只返回信息，不会下载视频。

支持的平台域名包括：

- `bilibili.com`、`b23.tv`
- `youtube.com`、`youtu.be`
- `x.com`、`twitter.com`
- `tiktok.com`、`douyin.com`
- `kuaishou.com`、`vimeo.com`

B 站链接会优先规范化为 `BV` 号链接。B 站回复包含标题、类型、UP、播放、弹幕、收藏、点赞、硬币、评论和简介；其他平台回复包含标题、作者、时长、简介和最佳流信息。

## 限制与清理

- JM PDF 下载大小上限为 150MB。
- 视频发送大小上限为 100MB。
- 同一个 JM 漫画 ID 的并发下载会串行执行，避免重复任务互相覆盖。
- JM 下载任务目录保留 24 小时，插件启动后每天清理一次过期目录。
- 视频下载临时目录会在 24 小时后清理。
- 文件、视频、合并转发发送能力取决于当前 AstrBot 平台适配器支持情况。

## 依赖

依赖声明在 `requirements.txt`：

- `jmcomic`
- `img2pdf`
- `PyYAML`
- `yt-dlp`
- `imageio-ffmpeg`

如果 JM 转 PDF 失败，请确认 `img2pdf` 已安装；如果视频解析或下载失败，请确认 `yt-dlp` 和 `imageio-ffmpeg` 已安装。

## 数据目录

运行时文件写入 AstrBot 数据目录：

```text
data/plugin_data/astrbot_plugin_lyn/
```

插件不会把 JM、Pixiv 或视频运行时产物写入源码目录。

## 常见问题

- Pixiv 发送失败：确认当前平台是 QQ OneBot v11，且机器人允许私聊触发用户。
- 视频下载失败：通常与目标平台风控、网络环境、链接失效或依赖缺失有关。
- 文件无法发送：确认平台适配器支持 `File` 或 `Video` 消息组件。
- 自动解析没有响应：确认消息不是命令，且链接域名在支持列表内。
